# Find Missing Number

A Python algorithm that finds all missing numbers in a sequence, from 1 up to the last element of the input list.

## Example

```python
[1, 2, 3, 5, 6, 7, 8, 9, 10, 11, 13, 14, 16]
# → [4, 12, 15]
```

## Technologies

- Python 3

## Getting Started

```bash
python code.py
```
